ProjValue Sum Tracker

Two quick steps to set this up. The chart will plot the sum of your ProjValue column, logged automatically every 5 minutes.

Step 1 — Add a Log tab to your Sheet

In your Google Sheet, click the + at the bottom to add a new tab. Name it exactly Log. Then put these two headers in row 1:

A1: Timestamp B1: ProjSum

That's it for the sheet. Leave the rest of the Log tab empty — the script will fill it in automatically.

Step 2 — Add the Apps Script

In your Google Sheet, go to Extensions → Apps Script. Delete any existing code and paste this in exactly:

function logProjSum() { var ss = SpreadsheetApp.getActiveSpreadsheet(); var dataSheet = ss.getSheetByName("Sheet1"); // ← change if your // tab has a diff name var logSheet = ss.getSheetByName("Log"); // Find ProjValue column by header var headers = dataSheet.getRange(1, 1, 1, dataSheet.getLastColumn()) .getValues()[0]; var col = -1; for (var i = 0; i < headers.length; i++) { if (String(headers[i]).toLowerCase().indexOf("proj") !== -1) { col = i + 1; break; } } if (col === -1) return; // couldn't find column // Sum the ProjValue column (skip header row) var lastRow = dataSheet.getLastRow(); if (lastRow < 2) return; var vals = dataSheet.getRange(2, col, lastRow - 1, 1).getValues(); var sum = 0; for (var j = 0; j < vals.length; j++) { sum += parseFloat(vals[j][0]) || 0; } // Write timestamp + sum to Log sheet logSheet.appendRow([new Date(), sum]); } function doGet() { var log = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName("Log"); var data = log.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }

Set the 5-min trigger: In Apps Script, click the clock icon (Triggers) in the left sidebar → + Add Trigger → Function: logProjSum → Event source: Time-driven → Type: Minutes timer → Every 5 minutes → Save.

Deploy the Web App: Click Deploy → New deployment → gear icon → Web app → Who has access: Anyone → Deploy → authorize when prompted → copy the URL.
